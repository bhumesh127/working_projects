
RAG with HuggingFace + FAISS Example
====================================

Setup Instructions:

1. Create virtual environment (optional but recommended)
   python -m venv venv
   source venv/bin/activate  (Linux/Mac)
   venv\Scripts\activate   (Windows)

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python rag_app.py

You can replace the sample documents list with:
- PDFs
- Database content
- Real-time APIs
- Vector store persistence

This is a simple real-time RAG pipeline example.
